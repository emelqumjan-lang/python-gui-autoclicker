# Modern Python Autoclicker 🚀

A fast, fully responsive, and beautifully designed Python autoclicker with a modern GUI. Thanks to multi-threading architecture, the interface never freezes or hangs during intensive clicking.

## ✨ Features
* **Modern UI:** Sleek dark-themed interface styled after Windows 11 using `CustomTkinter`.
* **Multi-threaded (No Freezes):** The clicking loop runs entirely in a background thread, keeping the GUI 100% responsive.
* **Global Hotkeys:** Start and stop clicking instantly using a hotkey (default is `F6`), even when you are in-game or the app is minimized.
* **Persistent Settings:** Automatically saves and loads your interval preferences and button configurations via `config.json`.
